# EDA-Business-Intelligence-Superstore-Task-2
# EDA & Business Intelligence Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Superstore dataset to identify business trends, sales patterns, and profitability insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* SQL
* Power BI / PowerPoint

## Key KPIs

* Total Sales: 2,297,200.86
* Total Profit: 286,397.02
* Total Orders: 5009
* Average Order Value: 229.86

## Key Insights

* Technology generated the highest sales.
* West region generated the highest profit.
* Discounts negatively impacted profitability.
* A small set of products generated a large portion of total revenue.
